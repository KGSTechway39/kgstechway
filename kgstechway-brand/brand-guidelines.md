## 2. Logo Mark — The Pathway Symbol

Three converging curved pathways merge into a single right-pointing arrow:

- **Top path (Red)** — Software Development: energy, power, foundation
- **Middle path (Orange)** — AI Solutions: intelligence, adaptability, warmth
- **Bottom path (Gold)** — CRM/ERP Services: value, reward, success

The convergence arrow represents unified direction — multiple technologies moving together toward one outcome: **business success**.

## 4. Typography

### Recommended Typeface

**Plus Jakarta Sans** (Google Fonts — free)
```
https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@300;400;800&display=swap
```

| Element | Weight | Size (in logo) | Letter-spacing |
|---------|--------|----------------|----------------|
| "KGS" | ExtraBold (800) | 31–34px | −0.8 to −1px |
| "techway" | Light (300) | 31–34px | +0.2 to +0.5px |
| SERVICES label | Regular (400) | 7.5–8px | +4.5 to +5.5px |
| Tagline | Light (300) | 8.5–9.5px | +1.0 to +1.2px |

### Fallback Stack
```css
font-family: 'Plus Jakarta Sans', 'Inter', 'Segoe UI', Arial, sans-serif;
```

## 5. Clear Space

Maintain clear space around the logo equal to the height of the letter **"K"** in the company name on all four sides. Never place text, images, or other graphic elements within this zone.

```
┌─────────────────────────────────────────┐
│              [K height]                 │
│  [K height]  LOGO HERE  [K height]      │
│              [K height]                 │
└─────────────────────────────────────────┘
```

---

## 7. Do's ✅

- Use the provided SVG files — they scale perfectly to any size
- Use the light variant on white, cream, or light-grey backgrounds
- Use the dark variant on navy, charcoal, dark-branded, or photographic backgrounds
- Use the icon-only mark for small placements (< 48px wide)
- Maintain the original colour of the pathway symbol in all uses
- Use the stacked variant when width is limited but height is available

---

## 8. Don'ts ✗

- **Do not** recolour the pathway symbol (red/orange/gold must stay warm)
- **Do not** stretch or distort the logo non-proportionally
- **Do not** add drop shadows, outlines, or effects to the wordmark
- **Do not** place the light logo on a dark background (use the dark variant)
- **Do not** use the logo at sizes smaller than the minimum sizes above
- **Do not** reproduce "KGS" and "techway" with different weights than specified
- **Do not** separate "KGS" from "techway" — they are a single compound name
- **Do not** change the tagline text

---

## 9. Logo on Coloured Backgrounds

| Background | Use variant | Notes |
|------------|-------------|-------|
| White / Light grey | Light variant | Default |
| Dark navy / Charcoal | Dark variant | Default |
| Brand orange (#E67E22) | Dark variant | Pathway colours still visible |
| Photography | Dark variant | Use semi-transparent overlay if needed |
| Gradient backgrounds | Test contrast | Prefer the dark variant |

---

## 10. Icon-Only Usage

The pathway mark may be used independently for:
- Browser favicons (16×16, 32×32, 48×48)
- App icons (iOS / Android) with solid brand-coloured background
- Social media profile avatars (add padding + background colour)
- Loading spinners or small UI elements

Recommended background for icon-only use: `#1A1A2E` (dark navy) or `#FFFFFF` (white).
